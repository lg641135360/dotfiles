# dotfiles
my config files

# tools
- shared
  - tmux
  - kitty
  - git
  - nvim
  - zsh(on ~/.config/zsh)
  - zed settings
- macintosh
  - aerospace[window manager]
- linux
  - xmonad[window manager]
  - xmobar[status bar]
  - dunst[message notify]
  - picom[animation composer]
  - rofi[like macos's alfred]
  - awesome[window manager]
  - Xresources[x11 font&color user config]

# how to use
```shell
chmod +x install.sh
./install.sh
```
